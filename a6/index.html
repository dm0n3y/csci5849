Instructions

Use the radio buttons above the calculator to choose a one-key or two-key switch control interface. The one-key interface utilizes the space bar to select the currently highlighted item. The two-key interface utilizes the space bar to navigate to the next item (whether this is a row or row item) and enter to select the highlighted item.
